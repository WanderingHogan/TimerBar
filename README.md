# TimerBar

Menu bar countdown timer to help with things you don't want to pollute your calendar with.

Created because there is free daycare that let me sit in the breakroom while they watch my kid, but a maximum of 1.5 hours, so I could work on things without setting up an alarm on my phone, or creating a calendar invite.

## Build Copy into Applicatons, & Run

```bash
./build.sh
cp -R "build/TimerBar.app" /Applications/
open /Applications/TimerBar.app
```
